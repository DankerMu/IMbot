# Proposal: Message Copy & Text Selection

## Problem

当前复制能力极其有限：

1. **仅有全量复制**：只能通过菜单 "复制全部输出" 一次性复制所有 agent 消息，无法复制单条消息。
2. **无文本选择**：用户无法在消息内容中选择特定文本进行复制。Compose `Text` 默认不可选。
3. **代码块复制孤立**：代码块有独立复制按钮，但普通文本消息没有。
4. **缺少消息级别操作菜单**：长按消息无任何响应——没有复制、分享、引用等操作。

## Reference

- **微信/Telegram**：长按消息弹出操作菜单（复制/转发/引用/删除）
- **ChatGPT Android**：消息下方有复制图标按钮
- **CodePilot**：消息 footer hover 显示 CopyButton（Check/Copy 图标切换，2s 反馈）

## Scope

仅涉及 Android 会话详情页的消息交互层。

## Success Criteria

- **单消息复制**：长按 agent 消息 → 弹出底部操作菜单 → "复制消息" 选项 → 复制纯文本到剪贴板 → toast 反馈
- **文本选择复制**：长按消息内文本 → 进入文本选择模式 → 系统选择手柄 → 选择后系统复制菜单
- **用户消息复制**：长按用户消息同样可复制
- **代码块复制保留**：现有代码块复制按钮不变
- **操作菜单可扩展**：底部菜单结构支持未来添加"分享"、"引用回复"等操作
