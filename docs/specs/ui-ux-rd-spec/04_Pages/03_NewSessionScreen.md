## и§„еҲҷ

- Step 1: provider йҖүжӢ©еҗҺиҮӘеҠЁзЎ®е®ҡ hostпјҲClaude/book вҶ’ MacBook, OpenClaw вҶ’ relay-localпјүгҖӮ
- Step 1: еҰӮжһң host offlineпјҢеҜ№еә” provider жҳҫзӨәдёә disabled + "зҰ»зәҝ" ж ҮзӯҫгҖӮ
- Step 2: book provider еҸӘиғҪеңЁ novel root дёӢжөҸи§ҲпјҲиҝҮж»Өе…¶д»– rootпјүгҖӮ
- Step 3: prompt дёҚиғҪдёәз©әгҖӮ
- Step 3: model dropdown йҖүйЎ№жқҘиҮӘ provider ж”ҜжҢҒзҡ„жЁЎеһӢеҲ—иЎЁгҖӮ
- "ејҖе§Ӣ" жҢүй’®зӮ№еҮ»еҗҺжҳҫзӨә loading overlayпјҢйҳ»жӯўйҮҚеӨҚжҸҗдәӨгҖӮ
