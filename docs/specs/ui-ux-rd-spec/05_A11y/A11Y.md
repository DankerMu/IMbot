# Accessibility (A11y)

## 目标

单用户产品，A11y 做基本保障，不追求 WCAG AA 全覆盖。

## 必须做

| Area | Requirement |
|------|-------------|
| Content description | 所有 Icon 有 `contentDescription` |
| Touch target | 最小 48dp × 48dp |
| Color contrast | Text / background 对比度 ≥ 4.5:1 |
| Screen reader | SessionCard、MessageBubble 可被 TalkBack 朗读 |
| Focus order | 合理 tab 顺序（TopBar → Content → BottomNav） |

## 组件级 A11y

| Component | Semantics |
|-----------|-----------|
| SessionCard | `"${provider} 会话, ${status}, ${prompt摘要}"` |
| StatusIndicator | `"状态: ${status}"` |
| ProviderChip | `"${provider}, ${selected ? '已选' : '未选'}"` |
| FAB | `"新建会话"` |
| InputBar send | `"发送消息"` |
| ToolCallCard | `"工具调用: ${toolName}, ${isRunning ? '执行中' : '已完成'}"` |
| ConnectionBanner | Live region, 状态变更时自动朗读 |

## 不做（MVP）

- 键盘导航优化（Android 手机，非重点）
- 高对比度模式
- 动画减弱模式（`prefers-reduced-motion`）
